# INFYNON CLI Docs

INFYNON CLI is a security-first command-line tool for AI-driven development.

It helps developers verify packages, scan dependencies, replay API workflows, trace repo context, and manage AI coding tasks from the terminal.

## Install

```bash
npm i -g infynon
```

## Main Commands

| Command | Use it for |
|---|---|
| `pkg` | Search packages, verify installs, scan dependencies |
| `weave` | Create, run, and replay API workflows |
| `trace` | Save repo context and implementation notes |
| `task` | Create and manage AI coding tasks |
| `coding` | Launch coding agents from INFYNON |
| `workspace` | Manage project workspaces |
| `soul` | Store stable user-level agent context |
| `doctor` | Diagnose installation issues |
