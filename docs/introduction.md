# Introduction

INFYNON CLI gives developers more control over AI-assisted development.

Modern AI coding tools can install packages, change files, generate API calls, and run commands quickly. INFYNON adds a control layer around that workflow.

Use INFYNON to:

- Verify packages before installing them
- Scan dependency files for known risks
- Test and replay API workflows
- Save context about repo changes
- Create tasks for AI coding agents
- Coordinate agent work across workspaces

INFYNON is designed for developers who want speed without blindly trusting every generated command.
