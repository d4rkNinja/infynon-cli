# Config Reference

INFYNON stores local user configuration and project state.

## User-level state

```txt
~/.infynon/
~/.infynon/ninja.yml
~/.infynon/workspaces/
~/.infynon/tasks/
~/.infynon/soul.md
```

## Project-level state

```txt
.infynon/
.infynon/.env
.infynon/trace/
```

## Notes

Do not delete these files unless you want to reset local INFYNON state.
