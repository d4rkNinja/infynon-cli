# Installation

Install INFYNON globally with npm:

```bash
npm i -g infynon
```

Check the installed version:

```bash
infynon --version
```

Check available commands:

```bash
infynon --help
```

## Requirements

- Node.js 18 or newer
- npm installed
- Terminal access
- Optional: Claude, Codex, Gemini, or other agent CLIs if you want to use `infynon coding`

## Other install methods

Go wrapper:

```bash
go install github.com/d4rkNinja/infynon-cli/go/cmd/infynon@latest
```

Linux and macOS:

```bash
curl -fsSL https://raw.githubusercontent.com/d4rkNinja/infynon-cli/main/install.sh | bash
```

Windows PowerShell:

```powershell
irm https://raw.githubusercontent.com/d4rkNinja/infynon-cli/main/install.ps1 | iex
```
