# Secure Package Install

Use INFYNON before installing dependencies generated by an AI tool or suggested by another developer.

## Steps

```bash
infynon pkg search <package>
infynon pkg npm install <package>
```

Other package managers:

```bash
infynon pkg uv add <package>
infynon pkg pip install <package>
infynon pkg cargo add <package>
```

## Strict mode

```bash
infynon pkg npm install express --strict high
infynon pkg npm install express --strict critical --no-input
```

## Notes

INFYNON adds a verification layer. It does not replace human review for high-risk dependencies.
